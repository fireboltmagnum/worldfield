WorldField

A cognitive architecture on a single shared latent space.

v0.1.0

Most AI systems break perception, memory, and reasoning into separate silos. WorldField asks: what if they all shared the same underlying "meaning space"?

Quick Start

pip install worldfield

# Or from source:
git clone https://github.com/fireboltmagnum/worldfield.git
cd worldfield
pip install -e .

# Launch the CLI:
worldfield
# or: python -m worldfield

Commands

worldfield                    # Interactive CLI
worldfield process "text"     # Process and store
worldfield query "text"       # Query without storing
worldfield refine "text"      # Run iterative refinement
worldfield stats              # Show system statistics
worldfield reset              # Reset engine state
worldfield benchmark          # Run benchmarks

Architecture

Every input — text, image, video — is encoded into a 128-dim shared latent space. Fragments are stored persistently in ChromaDB. Slot memory keeps multiple concepts active. A PMI/lift graph learns relations. Concept memory tracks confidence, uncertainty, temporal decay, and hierarchy.
